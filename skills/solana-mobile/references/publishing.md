## Usage notes

- Identify the submission type first: new app, update, Google Play migration, or web-app wrapper.
- Call out prerequisites early: signed APK, App NFT for first-time publishers, Release NFT for release submission, and any portal or CLI setup work.
- When the user asks compliance questions during publishing, also load [`policies.md`](policies.md).
